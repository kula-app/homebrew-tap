# Kula-app Tap

## How do I install these formulae?

`brew install kula-app/tap/<formula>`

Or `brew tap kula-app/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kula-app/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
